Privacy Policy for Makris Bot

Last updated: 2026

Makris Bot respects user privacy and only stores minimal data required for functionality.

1. Data Stored

  • Server IDs
  • User IDs (for moderation, logs, whitelist)
  • Channel IDs (for logs and configuration)

2. Data Usage

This data is used only for:

  • Moderation commands
  • Anti-Nuke protection
  • Logging server actions
  • Role and system configuration

3. Data Sharing

We do NOT sell, share, or distribute any data to third parties.

4. Data Control

Server owners can remove the bot at any time to stop data processing.

5. Contact

For concerns, contact the support server.

Using Makris Bot means you agree to this policy.
